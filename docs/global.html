identify(image, geometry)

Given an image and a point geometry, the identify function returns the pixel value of the image at the given point.
Parameters:
Name Type Description
image object the image is a raster representation derived from the getTiff() method of a GeoTIFF loaded using the geotiff npm package.
geometry string | object geometry can be an [x,y] array, a GeoJSON point object, or a string representation of a GeoJSON point object.
